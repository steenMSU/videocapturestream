Privacy Policy for FB Stream Capture

Last updated: April 19, 2024

Introduction

This app is for personal use only and processes Facebook livestream data with user permission. No personal data is collected or shared with third parties.

Data Collection

This application only accesses Facebook livestream data with explicit user authorization through Facebook's permission system. We process:

  • Video streams from Facebook pages you manage
  • Basic page information required to access these streams

Data Usage

Livestream data is processed only for the purpose of frame extraction and analysis, and is not shared with any third parties. Extracted video frames are stored securely in Google Cloud Storage.

Data Storage

We store:

  • Extracted video frames from your livestreams
  • Basic metadata about the original video source
  • No personal user information is stored beyond what is necessary for authentication

Data Retention and Deletion

We follow these data retention and deletion practices:

  • Extracted video frames are automatically deleted after 30 days
  • Authentication tokens are refreshed periodically and old tokens are discarded
  • You may request complete deletion of all your data at any time (see below)

Data Deletion Mechanisms

You have multiple options to delete your data:

  1. Email Request: Send a data deletion request to [Your Email]. Please include your Facebook Page ID in the request. We will process all deletion requests within 14 days and send confirmation when completed.
  2. Facebook Data Deletion Request: You can also submit a data deletion request through Facebook's systems. When we receive a Data Deletion Request from Facebook, we will delete all data associated with your Facebook ID within 14 days.
  3. Account Disconnection: If you disconnect our app from your Facebook page, we will automatically delete all associated data within 30 days.

User Rights

You have the right to:

  • Access your data
  • Request correction of inaccurate data
  • Request deletion of your data
  • Withdraw consent at any time

Data Security

We implement appropriate technical and organizational measures to protect your data, including encryption and access controls.

Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page.
